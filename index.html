Bem-vindo à Lua!

A Lua é uma linguagem de programação leve e flexível, projetada para ser usada em uma variedade de aplicações, desde jogos até sistemas embarcados.

Com sua sintaxe simples e fácil de aprender, a Lua é uma escolha popular para desenvolvedores de todos os níveis.

Sobre a Lua

A Lua foi criada em 1993 pelo Programa de Engenharia de Software da Pontifícia Universidade Católica do Rio de Janeiro (PUC-Rio) para ser uma linguagem de script para jogos.

Hoje, a Lua é amplamente utilizada em uma variedade de aplicações, desde jogos até sistemas embarcados, e é suportada por uma comunidade ativa.
